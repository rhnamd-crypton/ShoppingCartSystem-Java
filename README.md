# Shopping Cart System (Java)

A console-based shopping cart application developed using Java.  
This project demonstrates core Java concepts and Object-Oriented Programming.

## Features
- View available products
- Add products to cart
- Remove products from cart
- View cart items and total price
- Menu-driven console interface

## Technologies Used
- Java
- Java Collections (ArrayList)
- Object-Oriented Programming (OOP)

## How to Run
1. Open the project in Eclipse or any Java IDE
2. Run the `Main.java` file
3. Follow the menu options in the console

## Learning Outcomes
- Strong understanding of Java fundamentals
- Hands-on experience with OOP concepts
- Practical use of Java Collections
- Console-based application design
